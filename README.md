# 亚马逊爆款运营训练场

一个完全在本地浏览器运行的14天学习工具。内容以《爆款单品打造SOP》为框架，并按亚马逊运营场景重构。

## 启动

在 PowerShell 中运行：

```powershell
.\start.ps1
```

然后访问 `http://127.0.0.1:8770/`。

## 功能

- 14天亚马逊爆款运营课程
- 知识地图与学习进度
- 闪卡、随机追问和知识测验
- 2分钟口述倒计时与本地录音回放
- 五维表达自评与错题加权
- Pure Daily Care NanoSteamer公开经典案例
- 本地进度保存、重置和JSON导出

## 数据与合规

- 学习进度保存在浏览器 `localStorage`，键名为 `amazon-hit-learning-v1`。
- 录音只在当前页面内生成临时回放地址，不会持久化或上传。
- NanoSteamer案例区分公开事实与教学推演，不虚构未披露的广告、利润或库存数据。
- 亚马逊广告、评价和FBA规则可能变化，正式执行前应以Seller Central和Amazon Ads后台最新政策为准。
